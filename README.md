# Network Marketing Content Command Center

AI-powered content generator for network marketers. Generate 30 days of posts, captions, Reels scripts, recruiting messages, follow-up sequences, objection responses, team training content, and personal branding — in seconds.

## Built With
- Next.js 15 + TypeScript
- Tailwind CSS + shadcn/ui
- Template-based AI content generation

## Tools Included
1. **30-Day Content Pack Generator** — 135+ assets per pack
2. **Recruiting Message Generator** — 5 messages per scenario
3. **Follow-Up Sequence Builder** — Day-by-day sequences
4. **Objection Crusher** — 20 objections × 3 response styles
5. **Team Training Generator** — 6 content types
6. **Personal Brand Builder** — Full brand package
